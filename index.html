O que é HTML

(Voltar)

A linguagem HyperText Markup Language, ou HTML, é usada para criar páginas na internet. Usamos tags, como imagens, links e texto, para organizar as coisas na página. Algumas partes cruciais do HTML são:

Estrutura fundamental: Utilizamos a tag "HTML" para iniciar e concluir uma página HTML. As tags “HEAD” e “BODY" estão dentro disso. Para fazer uma página parecer bonita, colocamos o título da página e links no "head".

Linguagens de programação do lado do servidor

(Voltar)

São usadas para descrever aplicativos executados no servidor, em vez do navegador do cliente. Esses tipos de linguagens são essenciais para criar aplicativos e sites interativos e dinâmicos.

Linguagens de programação do lado cliente

(Voltar)

A linguagem de programação do lado do cliente é aquela usada no navegador do cliente, ou seja, no computador do usuário final. Essas linguagens são essenciais para manter as páginas da web dinâmicas e interativas, sendo JavaScript uma das melhores linguagens de programação do lado do cliente.
